# Customer_Behaviour_Analysis
Data Analytics Project Showcasing customer behaviour using MYSQL, Power BI and Python

## Overview

This project demonstrates a complete data analytics workflow—from loading raw data in Python to performing exploratory analysis, data cleaning, SQL querying, building a Power BI dashboard, and preparing a final business report and presentation.
It is designed to build a strong Portfolio Project for interviews and LinkedIn.

## Tools & Technologies

### Python (Pandas) – Data loading, EDA, and cleaning
### MySQL – Running analytical SQL queries
### Power BI – Dashboard design and visualization
